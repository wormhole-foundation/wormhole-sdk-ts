# Connect SDK Examples


## Usage

1) git clone this repo and rebuild then cd to this directory

```sh
git clone  https://github.com/wormhole-foundation/wormhole-sdk-ts.git
cd wormhole-sdk-ts
npm install
npm run build
cd examples
```

2) Optionally modify token bridge or cctp programs

3) run stuff

```sh
npm run tb
# or
npm run cctp
# or
npm run gateway
```
